# The Book Worm — Library Management System

## Run in VS Code

1. Open this folder in VS Code
2. Open Terminal (Ctrl + `)
3. Install Flask:
   pip install flask
4. Run the app:
   python app.py
5. Open browser: http://localhost:5000
